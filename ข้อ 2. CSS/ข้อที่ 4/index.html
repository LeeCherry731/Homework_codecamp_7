2. grid และ flex แตกต่างกันอย่างไร

  • Flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน
  • Grid เราสามารถจัดของไว้ตรงไหนก็ได้ใน grid layout เช่นตัวอย่างตามรูปด้านล่าง เราบอก grid ว่าให้ layout ของ header เต็ม row บรรทัดต่อมาให้ menu เป็น 2 columns content เป็น 8 columns อะไรประมาณนั้น

3. justify and align แตกต่างกันอย่างไร

  • justify-content: Horizontal Alignment & Spacing along primary axis (X-axis)
  • align-items: Vertical Alignment only along secondary axis (Y-axis)
